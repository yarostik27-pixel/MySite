<h2>Мій сайт RentCars UA</h2>
Макет сайту: https://www.figma.com/design/U8IjfcokYiF7wxuxsAPirv/Rent-Car---Landing-page--Community-?node-id=0-1&t=nflgGzL3okQxiJXx-1
<br>
Це сайт сервісу RentCars UA для оренди автомобілів в Україні.
<br>Його головна ідея: користувач може швидко знайти авто, вибрати місто, дату, тип автомобіля, переглянути популярні машини та натиснути кнопку оренди.
<br>Сайт відповідає за презентацію сервісу оренди авто, показ доступних автомобілів, пояснення процесу оренди та збір першої дії від користувача через кнопки пошуку або оренди.
<br>На сайті, відрображені відгуки користувачів
<br>Також на сайті, розтошовані такі застосунки Google Play та App Store, через які можно завантажити застосунок та орендувати машині, на будь якій вибір.
